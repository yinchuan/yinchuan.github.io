Run Roslaunch In Currenty Directory

Published: Thu 04 May 2023
By yinchuan

In Computing.

By default, node processes started by roslaunch are running in ~/.ros/. To see the current running directory of a process, use command pwdx <pid>.

It's sometimes useful to start the process in current directory. It can be done by setting ROS_HOME environment: ROS_HOME=. roslauch <pkg> <launch_file>. This command sets the running directory to current directory.

If you want to set the running directory to where the executable is, you can set attribute cwd=node in launch file. See documentation of <node> tag.
