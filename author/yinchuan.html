use Caps-Lock as Ctrl on Ubuntu 22.04

Published: Wed 26 April 2023
By yinchuan

In Computing.

why?

Ctrl is used very often in terminal so it's useful to have it closer. The benefit of using Caps-Lock as Ctrl is that the hand doesn't need to move a lot to reach it. The downside is Caps-Lock becomes further but it's rarely used so it's acceptable.

how?

After tried a few ways, I found this solution works for me.

  1. edit file /etc/default/keyboard, add ctrl:swapcaps to XKBOPTIONS="", so it looks like XKBOPTIONS="ctrl:swapcaps".
  2. run command sudo dpkg-reconfigure keyboard-configuration. choose keep the configuration in file when asked.
  3. reboot.

any other XKBOPTIONS available?

yes, I found a list of options and their meaning in xorg.lst.
